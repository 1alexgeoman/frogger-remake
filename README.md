# Frogger Game - README

## Overview:
This is a modern adaptaiton of the classic frogger game using JavaFx.

### Key Features:
- Dynamic obstacles
- Music
- Collision detection
- Score tracking
- Timer
- Lives system


### Project Structure:
- Model
- Controller
- View
- Resources

### How to play:
- Navigate using WASD
- Avoid vehicles and drowning the river
- Reach the safe zones within the time limit
  
### Key Classes
- `Main.java`: Entry point, sets up the game. 
- `Animal.java`: Main character logic and interactions. 
- `Obstacle.java`, `Log.java`, `Turtle.java`: Dynamic game objects. 
- `MyStage.java`: Game stage and background. 
- `FroggerUI.java`: Handles UI components like score and lives display.
